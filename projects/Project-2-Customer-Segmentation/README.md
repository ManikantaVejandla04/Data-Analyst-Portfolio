# Customer Segmentation Project Template

## Project Overview
This project aims to segment customers based on their purchasing behavior to improve marketing strategies and increase customer satisfaction.

## Contents
- Data Collection
- Data Cleaning
- Exploratory Data Analysis
- Modeling
- Results
- Conclusion

## Requirements
- Python 3.x
- Pandas
- Numpy
- Matplotlib
- Seaborn
- Scikit-learn

## Installation
```bash
pip install pandas numpy matplotlib seaborn scikit-learn
```

## Usage
1. Clone the repository:
   ```bash
   git clone https://github.com/ManikantaVejandla04/Data-Analyst-Portfolio.git
   cd Data-Analyst-Portfolio/projects/Project-2-Customer-Segmentation
   ```
2. Run the main script:
   ```bash
   python main.py
   ```
